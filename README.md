# Final-Project

This project uses machine learning to predict whether a loan applicant is likely to default based on historical financial data. It compares Logistic Regression and Random Forest models, with Random Forest achieving better performance after hyperparameter tuning. The workflow includes data cleaning, exploratory data analysis, feature engineering, model training, and evaluation using accuracy, precision, recall, and F1-score. The final model helps improve loan risk assessment by identifying high-risk borrowers more effectively.

# Workflow
Data cleaning and preprocessing
Exploratory data analysis (EDA)
Feature engineering
Model training (Logistic Regression & Random Forest)
Model evaluation
Hyperparameter tuning
Final model selection

# Key Results
Random Forest outperformed Logistic Regression in overall performance
Improved recall for identifying high-risk borrowers
Tuned model showed better generalization on unseen data

# Technologies Used
Python
Pandas & NumPy
Scikit-learn
Matplotlib / Seaborn
Google Colab

# Conclusion
This project demonstrates how machine learning can be used to support financial risk assessment. The Random Forest model provides a more accurate and reliable approach for predicting loan defaults.
